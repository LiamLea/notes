### Overview

#### 1.Storage Class

| Class | Use Case | Retrieval | $/GB-mo (us-east-1) | Min Duration |
|---|---|---|---|---|
| **Express One Zone** | Latency-sensitive, high-throughput, single AZ | 1–5 ms | $0.16 | — |
| **Standard** | Frequently accessed data | ~ms | $0.023 | — |
| **Intelligent-Tiering** | Unknown/changing access patterns | ~ms | $0.023 / $0.0125 / $0.004 | — |
| **Standard-IA** | Infrequent access, rapid retrieval | ~ms | $0.0125 | 30 days |
| **One Zone-IA** | Infrequent, non-critical, single AZ | ~ms | $0.01 | 30 days |

| **Glacier Deep Archive** | Long-term retention (7–10 yr) | 12–48 hr | $0.00099 | 180 days |

#### 2.Access Control

##### (1) ACL (leagcy)

* `block_public_acls = true` 
  * don't allow to create ACL allowing public access, e.g.
    * `aws s3 cp index.html s3://qor5-test-ciam-console-frontend/ --acl public-read`

* `ignore_public_acls = true`
  * ignore any public ACLs that already exist i.e. public acl not works

##### (2) Bucket Policy (modern)
* `block_public_policy = true`
  * don't allow to create bucket policy allowing public access

* `restrict_public_buckets = true`
  * ignore any public policy that already exist i.e. public policy not works

#### 3.Access Control For Cloudfront

##### (1) Origin Access Identity (OAI, legacy) 
```tf
# 1. Create the OAI resource
resource "aws_cloudfront_origin_access_identity" "example" {
  comment = "OAI for my S3 bucket"
}

# 2. Attach it to the CloudFront Distribution
resource "aws_cloudfront_distribution" "s3_distribution" {
  origin {
    domain_name = aws_s3_bucket.example.bucket_regional_domain_name
    origin_id   = "myS3Origin"

    s3_origin_config {
      origin_access_identity = aws_cloudfront_origin_access_identity.example.cloudfront_access_identity_path
    }
  }
  # ... other configuration ...
}

# 3. Update the S3 Bucket Policy to allow the OAI Canonical User ID
resource "aws_s3_bucket_policy" "allow_access_from_oai" {
  bucket = aws_s3_bucket.example.id
  policy = jsonencode({
    Version = "2012-10-17"
    Statement = [
      {
        Action   = "s3:GetObject"
        Effect   = "Allow"
        Resource = "${aws_s3_bucket.example.arn}/*"
        Principal = {
          AWS = aws_cloudfront_origin_access_identity.example.iam_arn
        }
      }
    ]
  })
}
```

##### (2) Origin Access Control (OAC)
```tf
# 1. Create the OAC resource
resource "aws_cloudfront_origin_access_control" "example" {
  name                              = "s3-oac-example"
  description                       = "OAC for S3 bucket"
  origin_access_control_origin_type = "s3"
  signing_behavior                  = "always"
  signing_protocol                  = "sigv4"
}

# 2. Attach it to the CloudFront Distribution
resource "aws_cloudfront_distribution" "s3_distribution" {
  origin {
    domain_name              = aws_s3_bucket.example.bucket_regional_domain_name
    origin_access_control_id = aws_cloudfront_origin_access_control.example.id
    origin_id                = "myS3Origin"
  }
  # ... other configuration ...
}

# 3. Update the S3 Bucket Policy to allow the CloudFront Service Principal
resource "aws_s3_bucket_policy" "allow_access_from_cloudfront" {
  bucket = aws_s3_bucket.example.id
  policy = jsonencode({
    Version = "2012-10-17"
    Statement = [
      {
        Action   = "s3:GetObject"
        Effect   = "Allow"
        Resource = "${aws_s3_bucket.example.arn}/*"
        Principal = {
          Service = "cloudfront.amazonaws.com"
        }
        Condition = {
          StringEquals = {
            "AWS:SourceArn" = aws_cloudfront_distribution.s3_distribution.arn
          }
        }
      }
    ]
  })
}
```
